# SADX TGS Mode by x-hax

TGS Mode is a mod for Sonic Adventure DX PC (2004) that adds features from the demo version of the game on the Dreamcast shown at Tokyo Game Show in 1998.

**Configuration**

- `Enable TGS Mode` activates the demo mode with a warped title screen, Tikal music and the old character select menu.

- `Enable 'Now Loading'` enables the SA1 logo loading screen used in the Autodemo and in TGS mode. This screen will replace all black fadeouts in the game, as well as stage title cards.

**Features of the TGS Mode**
- The title screen is fliped horizontally, which was apparently intentional.
- Hold X+Y on the title screen to activate the debug Level Select menu. Use L/R and the analog stick to navigate the menu.
- Normally the game starts in story mode. Hold A+B on the last character tutorial screen to start the game in demo mode.
- Hold X+Y and press Start on the character select screen to activate the secret debug mode (not sure if it still works).
